Після запуску Python термінало(коли вперше була натиснута кнопка запуску програми), щоб підключитись до
віртуальної машини треба ввести команду: .\env\Scripts\activate.bat

Після чого ніби нічого не змінилось, але щоб перепровірити себе, можна ввести команду:
pip freeze
Дана команда надає список всіх бібліотек встановлених в віртуальне оточення.

Для запуску WEB-сторінки ввети команду: python app.py
